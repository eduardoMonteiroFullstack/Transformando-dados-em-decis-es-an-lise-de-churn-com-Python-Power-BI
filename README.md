O projeto começou com a leitura e tratamento de uma base CSV utilizando Python, realizando:
✔️ limpeza de dados
✔️ tratamento de valores nulos
✔️ padronização de colunas
✔️ conversão de tipos para análise

Após o tratamento, exportei a base para Excel e segui para o Power BI, onde construí medidas e indicadores importantes para análise do comportamento dos clientes.

📊 Alguns insights encontrados:
• Clientes com contratos de 2 anos possuem cerca de 97% de chance de permanecer ativos
• O método de pagamento via cheque eletrônico apresentou a maior taxa de desistência
• A taxa geral de churn identificada foi de 26,58%
• O dashboard permitiu visualizar padrões importantes relacionados a contrato, faturamento e retenção

🛠️ Tecnologias utilizadas:
🐍 Python
📑 Pandas
📊 Power BI
📁 Excel

Projetos como esse mostram como a análise de dados pode gerar insights relevantes para tomada de decisão e estratégias de negócio.
